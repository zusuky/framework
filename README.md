# Zusuky framework  

## サンプルアプリの導入方法
以下が機能があるサンプルアプリ導入を導入する方法でう。
- ユーザ登録
- ログイン

<br>

### 1-1. アプリケーションの上書き
sample.zipを解凍し、解凍ファイルに含まれるappフォルダの内容で、appフォルダを上書する。

<br>

### 1-2. テーブルの作成
CREATE_TABLE.ddlに記述されているSQLを実行し、テーブルを作成する。

<br>

### 1-3. サンプル表示
フロント画面用のユーザ登録確認

```
http://localhost
```

管理画面用ユーザの登録・ログイン確認

```
http://localhost/admin/
```
