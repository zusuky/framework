# Zusuky framework  
<br>
## サンプルアプリの導入の仕方
以下が可能なサンプルアプリを導入できます。
- ユーザ登録
- ログイン
<br>
## サンプル導入方法

### 1-1. アプリケーションの上書き

sample.zipを解凍し、解凍ファイルに含まれるappフォルダの内容で、appフォルダを上書する。

### 1-2. テーブルの作成

CREATE_TABLE.ddlに記述されているSQLを実行し、テーブルを作成する。

### 1-3. サンプル表示

フロント画面用のユーザ登録確認

```
http://localhost
```

管理画面用ユーザの登録・ログイン確認

```
http://localhost/admin/
```
